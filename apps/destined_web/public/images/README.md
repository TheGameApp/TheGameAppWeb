# Images Directory

This directory contains all image assets for the Destined web application.

```

│
├── icons/                  # Icon assets
│   ├── star.svg           # Star icon (logo)
│   ├── moon.svg           # Moon icon
│   ├── star-decoration.svg # Decorative star
│   ├── moon-decoration.svg # Decorative moon

```

## Image Guidelines

### Hero Images
- **Format**: PNG with transparency
- **Dimensions**: 1080x2340px (phone mockups)
- **Purpose**: Showcase app interface

### Icons
- **Format**: SVG (preferred) or PNG
- **Size**: 24x24px to 64x64px
- **Style**: Cosmic/mystical theme (stars, moons, celestial)

### Logo
- **Format**: PNG with transparency
- **Sizes**: 
  - Icon: 512x512px
  - Logo: Variable width, 80px height

### Store Badges
- **Format**: SVG or PNG
- **Dimensions**: Official store badge dimensions
- **Source**: Download from official sources

### Testimonials
- **Format**: JPG or PNG
- **Dimensions**: 200x200px (square)
- **Style**: Professional headshots

- Purple: #9d4edd

## Optimization

- Compress images using tools like TinyPNG or ImageOptim
- Use WebP format for better performance
- Provide @2x and @3x versions for retina displays
- Use lazy loading for images below the fold
